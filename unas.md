# Required setting at your store
First you will need to go `Alapbeállítások`

![basicsettings_dropdown](https://user-images.githubusercontent.com/67690333/166472210-66a5ad66-9cd6-49de-8a4d-244956a8955d.png)

Then click `Működés` at the top

![menu](https://user-images.githubusercontent.com/67690333/166472471-0c453d02-147c-46fa-a115-e04f92c901e8.png)

Then scroll down to `Termékkel kapcsolatos` segment of the page

![segment](https://user-images.githubusercontent.com/67690333/166472924-698a9a1b-3bfc-4862-93b4-b3a33c57262b.png)

And turn off the following option 

![switch](https://user-images.githubusercontent.com/67690333/166473083-d6fb5679-ad04-476b-b243-0275b19eb170.png)

If you do not do this, the following error will be produced at the next steps.

![error](https://user-images.githubusercontent.com/67690333/166473565-a1deed51-51fc-44d1-8f64-9f66d410cc0b.png)

# The required files
The produced excel file by the program, what can be used at unas's item management page to upload the category and item database!
The produced zip files, if you want to upload item pictures as well.

## Upload your item database

This menu can accessed under

![dropdown_menu](https://user-images.githubusercontent.com/67690333/166469468-3a0a2cb1-3207-421c-b001-8f80284ded32.png)

After clicking the above shown menu. You will need to select from four options to upload your database with the produced data.

![import_item_database](https://user-images.githubusercontent.com/67690333/166469637-222d8cf2-ab82-4274-87ac-66dcfcce4a30.png)

If you are upload data to a blank store select the first option
![do_not_delete_anything](https://user-images.githubusercontent.com/67690333/166470211-3f179e3f-e4aa-4bf9-8eea-cfdd11d5fecb.png)

If you had a previous category structure select the third option
![delete_item_and_category](https://user-images.githubusercontent.com/67690333/166470550-d96ea369-fc88-4a6f-90d0-8799388e7c9f.png)

## Upload the required excel file
If you selected the right option for you. You need to select the excel file that had the program produced.

  > This file is always named by the following scheme `adatok_YEAR-MONTH-DAY-HOUR-MINUTE-SECOUNDS.xlsx`

![upload_file](https://user-images.githubusercontent.com/67690333/166470769-5b395c33-0529-4f78-9e46-603f6f89970a.png)

Then click on the `Feltöltés` button to start data import.
![upload_button](https://user-images.githubusercontent.com/67690333/166473908-d56d5099-fa3e-4802-adc1-33e1cf05996a.png)
